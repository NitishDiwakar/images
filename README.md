📸 Image Manager

Lightweight, file-based image viewer + manager (no database).

- 📂 Organize images in directories
- ⚡ Fast fullscreen viewer with keyboard, mouse & swipe navigation
- 🔄 Lazy loading for smooth scrolling
- 🆕 Latest images shown first
- 🛠 Simple management panel ("/manage")
- 📤 Multi-image upload, rename, delete, create directories
- 🔐 Works with ".htaccess" or PHP session auth
- 📱 Mobile-friendly, clean UI
